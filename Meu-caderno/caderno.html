O que é o HTML

HTML (Hypertext Markup Language) é uma linguagem de marcação utilizada para páginas web e aplicativos,
é formado por elementos. Os elementos por sua vez devem consistir em tag de abertura e fechamento (<) e (<(/)), o conteúdo em si fica no espaço entre as tags. Além disso, elas podem receber atributos.

Listas

Há duas formas de criarmos uma lista no mundo HTML. A primeira é a lista desordenada (ou Unordered List) e a segunda é a lista ordenada (ordered list).
A lista desordenada corresponde ao elemento 'ul' (Unordered List).
A lista ordenada corresponde ao elemento 'ol' (Ordered List)
